Meteor404

A Rails App using the NASA Meteorite Landings API and Google Maps API to Give Directions to the Nearest Unfound Meteorite

NASA's Meteorite Landings API has information from The Meteoritical Society on every known meteorite landing in history.

With over 34,000 known impacts on record, meteorites are categorized as either "found" or "fallen". This app focuses on the unfound meteorites, or those that have a known estimated location, but nobody has gone and found a space rock there yet.

Our app harnesses the NASA meteorite data available and uses the Google Maps API to show you all estimated meteorite landings near your location and provides information on them such as the meteorite's name and year of impact. You can also access a specific meteorite and then receive driving directions to its provided location.

Technologies

Ruby on Rails Front and Back End, PostgreSQL Database, NASA Meteorite Landings API, Google Maps API
